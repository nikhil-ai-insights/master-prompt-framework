# ⚙ METHODS

The Master Prompt Framework follows a structured methodology:

### 1️⃣ Role Definition
Clearly define the AI's role and expertise level.

### 2️⃣ Objective Clarity
State exactly what outcome is expected.

### 3️⃣ Context Injection
Provide necessary background information.

### 4️⃣ Constraints and Rules
Define boundaries, limitations, and formatting rules.

### 5️⃣ Output Structure
Specify how the response should be formatted.

### 6️⃣ Tone and Style Control
Control writing style, depth, and audience level.

### 7️⃣ Example Guidance (Optional)
Provide examples to improve consistency.

The included Master Prompt Template integrates all these components into a reusable structure.
